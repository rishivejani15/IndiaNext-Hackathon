## 🚀 Overview

**KAVACH (Digital Armour)** is an AI-powered cybersecurity platform designed to detect, analyze, explain, and mitigate modern cyber threats in real time.

Unlike traditional security tools that only generate alerts, KAVACH provides explainable threat intelligence, personalized risk assessments, psychological vulnerability analysis, prompt injection detection, phishing analysis, steganography detection, deepfake detection, and incident management through a unified platform.

Built for organizations, security teams, and everyday users, KAVACH transforms complex cybersecurity threats into actionable insights.

# 🔄 System Workflow

1. User submits content through dashboard, extension, scanner, or detector.
2. Inputs are processed by specialized detection engines.
3. Each engine produces an independent threat score.
4. Fusion Engine combines results using weighted risk scoring.
5. XAI Engine generates explanations and impact simulations.
6. Safe Word Challenge is triggered for high-risk events.
7. Incident Dashboard logs and visualizes findings.
8. Threat intelligence is enriched through ThreatPulse.

# 🔐 Safe Word Challenge

When risk score exceeds **0.70**, KAVACH triggers a human verification mechanism.

Purpose:

* Prevent social engineering attacks
* Detect account compromise
* Verify user authenticity
* Reduce automated exploitation
